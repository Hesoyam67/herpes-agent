## Launch Post Draft

I forked Hermes Agent into Herpes Agent because autonomous agents deserve a little OSHA violation.

It is open-source, free-first, and built around a stupid but useful premise:

- ship tiny autonomous workflows
- turn lessons into skills
- mutate fast
- spend basically nothing
- make agent tooling less corporate beige

Repo: https://github.com/Hesoyam67/herpes-agent

Current infection vectors:
- neon `herpes-agent` CLI skin
- autonomous growth skill
- roadmap + content queue
- zero-budget chaos doctrine

Star it if you want to watch an AI agent become a productivity rash.
